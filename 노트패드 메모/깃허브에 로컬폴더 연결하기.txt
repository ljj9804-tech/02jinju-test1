#깃허브에 신규 레포지토리 만들기

깃허브 접속하여 로그인

좌측 상단 메뉴 선택 > 레포지토리(Repositories) 선택

신규 레포지토리(New repository) 클릭

레포지토리 이름 입력 후 레포지토리 생성(Create repository) 클릭


#로컬파일 깃허브에 연동시키기

로컬 라이브러리에 신규폴더 생성하기

신규 생성한 레포지토리로 돌아가 command line 코드 복사하여
"git add ."
코드 추가하여 복사, 신규폴더 cmd 열어서 붙여넣기

*참고* 로컬을 깃허브에 처음 등록할 경우 본인 계정을 1차 등록해야되는데
  git config --global user.email "you@example.com"
  git config --global user.name "Your Name"
요청하는 코드에 자기 정보 채워서 돌려주면 됨
(그럼 깃허브에서도 함 더 물어볼텐데 그것도 ㅇㅋ 하면됨)

모든 과정을 끝내고 내 깃허브에 들어가 신규 레포지토리창에서
새로고침 하면 이렇게 연결된거 볼 수 있음

이제 vscode에 신규폴더 열어 작업 후 푸시해보기!
